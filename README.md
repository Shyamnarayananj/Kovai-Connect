# Kovai Connect

Kovai Connect is a digital onboarding platform designed to help local businesses in Coimbatore (Kovai) establish and manage their digital presence more efficiently. The platform focuses on simplifying the onboarding process for businesses while providing a modern, scalable, and user-friendly interface.

## Features

* Digital onboarding workflow for local businesses
* Modern and responsive user interface
* Fast performance powered by Vite and Bun
* Type-safe development using TypeScript
* Scalable frontend architecture
* Clean code formatting and linting setup
* Optimized developer experience

## ⚙️ Installation

Clone the repository:

```bash
git clone https://github.com/Shyamnarayananj/Kovai-Connect.git
```

Move into the project directory:

```bash
cd Kovai-Connect
```

Install dependencies:

Using npm:

```bash
npm install
```

Or using Bun:

```bash
bun install
```

---

## Running the Project

Start the development server:

Using npm:

```bash
npm run dev
```

Using Bun:

```bash
bun run dev
```

The application will start on the local development server.

---

## Build for Production

```bash
npm run build
```

Or:

```bash
bun run build
```

---

## Linting

Run ESLint:

```bash
npm run lint
```
